# SimpleMotion Global Logos

Brand logos and visual identity assets for the SimpleMotion enterprise.

## Logo Variants

| Folder | Brand | Subtitle | Usage |
|--------|-------|----------|-------|
| `sm/` | SimpleMotion | (none) | General corporate use |
| `sp/` | SimpleMotion | PROJECTS | Project documentation, invoices |
| `sa/` | SimpleMotion | ARCHITECTURE | Architecture division documents |

## Available Files

### SM (SimpleMotion)
- `sm-black-banner.svg` - Horizontal logo, black text, no subtitle

### SP (SimpleMotion Projects)
- `sp-black-banner.svg` - Horizontal logo, black text with "PROJECTS" subtitle

### SA (SimpleMotion Architecture)
- `sa-black-banner.svg` - Horizontal logo, black text with "ARCHITECTURE" subtitle

## Usage in Documents

Reference logos in HTML documents using relative paths:

```html
<img src="../12-SimpleMotion/sm-black-banner.svg" width="55%">
```

For project documents, copy the appropriate logo to the project's `10-Engage/12-SimpleMotion/` folder.

## Color Variants

- **black** - Black text on transparent background (for light backgrounds)
- **white** - White text on transparent background (for dark backgrounds)
